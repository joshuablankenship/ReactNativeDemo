# ReactNativeDemo
```
In order to the app first make a .env and add the following environment variables
to configure your ouath provider of choice.
GOOGLE_OAUTH_APP_GUID,
ISSUER,
CLIENT_ID,
REDIRECT_UR,
Then run
`npm install`
Then navigate to the \ios\ directory and run
`pod install`
then run
`npm ios`
in the main application directory
```
